    Project Overview

    A module used to make basic calculations.

    • add(a: float, b: float) -> float: Return the sum of two numbers.
    • subtract(a: float, b: float) -> float: Return the difference of two numbers.
    • multiply(a: float, b: float) -> float: Return the product of two numbers.
    • divide(a: float, b: float) -> float: Return the quotient of two numbers.
    • power(base: float, exponent: float = 2) -> float: Return base to the power of exponent.
    • sqrt(a: float) -> float: Return the square root of a.
